# Expression Inception \(Technically A Loop\)

## Expressions

Expressions Are Used With Basic Values Using \(/, +, -, \*\).

Values Can Be Loose \(selfed\) Or Requested. \(tokenized\)

Here is a Shell Version of a Expression

```
$ 1 + 3 && 3 * 3 && 2 / 1 && 3 - 7
```
